# 미션 - 자동차 경주 게임

## 🔨 기능 목록

### 1. 경주할 자동차 이름을 , 구분으로 입력받는다.
### 2. 시도할 횟수를 입력받는다.
### 3. 시도한 횟수 만큼 자동차들이 `-`로 전진하거나 이동을 하지 않는다.
- 0에서 9사이의 Random 값이 4이상이면 전진한다.
### 4. 가장 많이 전진한 자동차가 승리
- 2명 이상일 수 있다

<br>

## ✍🏻 입출력 요구사항
### ⌨️ 입력
- 경주 할 자동차 이름(이름은 쉼표(,) 기준으로 구분)
```
pobi,woni,jun
```
- 시도할 회수
```
5
```

### 🖥 출력
- 각 차수별 실행 결과
```
pobi : --
woni : ----
jun : ---
```
- 단독 우승자 안내 문구
```
최종 우승자: pobi
```
- 공동 우승자 안내 문구
```
최종 우승자: pobi, jun
```
- 예외 상황 시 에러 문구를 출력해야 한다. 단, 에러 문구는 [ERROR] 로 시작해야 한다.
```
[ERROR] 시도 횟수는 숫자여야 한다.
```

```

시도할 회수는 몇회인가요?
```
